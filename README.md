# FFmpeg-X264-Encode-for-iOS
利用FFmpeg+x264将iOS摄像头实时视频流编码为h264文件

本工程支持armv7，arm64，i386, x86_64架构，iOS最低版本7.0。

FFmpeg采用的是目前最新的版本version 2.7.2
x264采用的版本是version x264-snapshot-20140930-2245.tar.bz2
